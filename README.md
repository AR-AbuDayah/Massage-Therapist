# Introduction

This project represents a sample system for managing massage services, customer profiles, scheduling, and service tracking.

---

# Actors

## Customer
Requests massage appointments and views available services.

## Receptionist
Handles appointment booking and customer records.

## Admin
Manages service types, staff, pricing, and system settings.

---

# Functional Requirements

## User Login System
Provides secure login for Admin and Receptionist.

## Appointment Booking
Allows scheduling, editing, and canceling sessions.

## Customer Database
Stores customer details, history, and preferences.

## Service Management
Admin can add, edit, or remove massage service types.
